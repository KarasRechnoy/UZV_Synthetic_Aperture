# SAR-стенд 40 кГц — конвейер обработки

Формирование радиолокационно-подобного изображения сцены ультразвуком
(несущая 40 кГц) методом синтезированной апертуры. Антенна (Tx + линейка Rx)
движется по прямой; набор пингов когерентно фокусируется time-domain
backprojection. Координаты целей для построения изображения не нужны.

## Структура

```
src/sar_lab/
  config.py             константы стенда, c(T), dx, λ
  io.py                 загрузка .npy и ground-truth YAML
  preprocessing.py      вычитание стационарного фона
  range_compression.py  согласованный фильтр (CW/ЛЧМ) + аналитический сигнал
  windows.py            окна апертуры (hamming/taylor/…)
  backprojection.py     когерентное суммирование (моно/бистатика, t0)
  autofocus.py          энтропия/контраст + автофокус c (метод Брента)
  calibration.py        ось дальности, оценка c/t0, поиск пиков
  visualize.py          изображение в дБ-шкале
scripts/
  convert_txt.py        сырой txt стенда -> .npy
  run_pipeline.py       полный конвейер: .npy -> сфокусированное изображение
tests/                  pytest на синтетике (одна точечная цель)
```

## Установка

```bash
pip install -e .[dev]
```

## Использование

1. Конвертация сырых данных:

```bash
python scripts/convert_txt.py raw/scene.txt  data/raw/scene.npy
python scripts/convert_txt.py raw/empty.txt  data/raw/empty.npy
```

2. Построение изображения (полный конвейер, автофокус + бистатика):

```bash
python scripts/run_pipeline.py \
    --scene data/raw/scene.npy --empty data/raw/empty.npy \
    --pulse chirp --band 38e3 42e3 --dur 1e-3 \
    --autofocus --bistatic --t0-ms 2.4 \
    --out results/scene
```

Наклонная антенна (CW 40 кГц / 25 мкс, наклон 30°):

```bash
python scripts/run_pipeline.py \
    --scene data/raw/scene_a30.npy --empty data/raw/empty_a30.npy \
    --pulse cw --f0 40e3 --dur 25e-6 \
    --autofocus --bistatic --tilt-deg 30 --t0-ms 2.4 \
    --out results/a30
```

Результат: `results/<...>/image.png` (дБ-шкала) и `image.npy` (комплексное).

## Алгоритм (кратко)

1. **Фон.** `S_clean = S_scene − S_empty` — убирает прямой сигнал и
   стационарный клаттер.
2. **Range compression.** Согласованная фильтрация строки в частотной
   области: `S_rc = IFFT(FFT(S)·conj(FFT(p)))`; затем преобразование
   Гильберта → комплексный (passband) сигнал.
3. **Backprojection.** Для каждого пикселя суммируются вклады всех пингов
   по гиперболе задержки `τ = D/c + t0` (D — полный путь Tx→цель→Rx, t0 —
   постоянная задержка тракта). Фаза несущей уже в сигнале — отдельная
   демодуляция не нужна.
4. **Автофокус.** `c` подбирается минимизацией энтропии изображения
   (метод Брента), без знания координат.
5. **Калибровка дальности.** Постоянная `t0` (аппаратная задержка
   триггера) сдвигает ось дальности к истинным значениям; при наличии
   одного эталонного отражателя калибруется точно.
6. **Визуализация.** Амплитуда в дБ относительно пика, отсечка по
   динамическому диапазону (≈28 дБ — иначе видны боковые лепестки
   сильных целей).

## Замечания

- Разрешение: азимут `δx ≈ λR/(2·L_SA)`, дальность `δR ≈ c/(2B)`
  (CW: B≈1/τ_pulse; ЛЧМ: B = полоса).
- «Полосы» на изображении сильных уголков — боковые лепестки PSF, а не
  расфокус; подавляются окном Тейлора и выбором динамического диапазона.
- Фокусная `c` (по энтропии) и дальностная `c` могут отличаться:
  абсолютная дальность опирается на калибровку `t0`.

## Тесты

```bash
pytest -q
```

Синтетика: одна точечная цель в известной позиции; проверяется положение
пика после range compression и backprojection и азимутальное разрешение
относительно теории.
